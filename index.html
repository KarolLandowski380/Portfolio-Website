O mnie

Nazywam się Karol i prowadzę działalność pod marką LANDEV jako Web Developer. Specjalizuję się w tworzeniu stron internetowych, głównie w oparciu o WordPress. Od 5 lat zajmuję się tym profesjonalnie, współpracując z różnymi firmami zarówno w formie kontraktów B2B, jak i z klientami indywidualnymi. Posiadam doświadczenie w projektach wymagających komunikacji zespołowej oraz z klientami w języku angielskim. Jestem otwarty na różne formy współpracy.

WordPress

Tworzę szablony od zera i modyfikuję istniejące motywy. Buduję strony przy użyciu różnych page builderów, dostosowując rozwiązanie do budżetu projektu i zapotrzebowania klienta. Posiadam doświadczenie w pracy z WooCommerce.

PHP

Wykorzystuję PHP do rozwoju funkcjonalności WordPress – tworzenia custom post types, szablonów oraz prostych wtyczek. Czasami pracuję z silnikiem szablonów Twig. Często korzystam z Advanced Custom Fields w projektach.

JavaScript

Dodaję interaktywność do stron przy użyciu vanilla JavaScript oraz lekkich bibliotek przyspieszających rozwój projektów. Posiadam umiejętności pracy z biblioteką GSAP do animacji.

CSS

Tworzę responsywne i nowoczesne style CSS z dbałością o szczegóły i optymalizację wydajności.

SCSS

Preferuję SCSS za możliwość organizacji kodu w bardziej przejrzysty sposób oraz wykorzystanie zmiennych i mixinów przyspieszających proces stylowania.

GIT

Pracuję z systemem kontroli wersji Git oraz GitHub. Mam doświadczenie w pracy z workflow produkcja/staging w projektach zespołowych.

FIGMA

Sprawnie obsługuję Figmę przy implementacji projektów graficznych oraz ekstraktowaniu zasobów potrzebnych do kodowania interfejsów użytkownika.
