O mnie

Nazywam się Karol i działam pod marką LANDEV jako Web Developer. Specjalizuję się w tworzeniu stron internetowych - głównie opartych na WordPressie. Zajmuję się tym profesjonalnie od 5 lat i pracuję oraz pracowałem z różnymi firmami w ramach kontraktów B2B jak i również z Klientami indywidualnymi. Mam doświadczenie w projektach, gdzie konieczna była komunikacja z zespołem oraz z Klientem w języku angielskim. Nie wykluczam innych form współpracy niż B2B.

WordPress

Tworzę szablony od zera i umiem modyfikować istniejące ale również buduję strony przy użyciu przeróżnych page builderów (zależnie od budżetu projektu i zapotrzebowania). WooCommerce nie jest mi obcy.

PHP

Wykorzystuję do tworzenia szablonów, post typów, prostych pluginów. Czasami piszę w Twigu. Często korzystam z ACF. Ekspertem w PHP nie jestem ale zawsze się odnajdę i dowiozę jak coś muszę w nim zrobić.

JavaScript

Nie straszne mi dodawanie interaktywności do strony przy użyciu JSa. Lubię wykorzystywać lekkie biblioteki do przyspieszenia pracy nad stronami. Potrafię w GSAP.

CSS

Bardzo lubię pisać CSS i uważam, że jestem w tym bardzo dobry.

SCSS

Jeszcze bardziej niż CSS lubię pisać SCSS bo pisze się go szybciej i jest on bardziej czytelny.

GIT

Przy niektórych projektach zdarza mi się pracować z Gitem oraz z GitHubem. Miałem wielokrotnie styczność z pracą w systemie produkcja/staging.

FIGMA

Projektować w Figmie nie potrafię bo się tym nie zajmuję ale wiem jak ją obsługiwać przy cięciu gotowego projektu.
